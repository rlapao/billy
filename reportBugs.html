Report Bugs

For issues with the Billy application you should submit your bug report to the Billy Issue tracker

Rules to bug reporting

  • 1. You should check if the bug has already been reported, to avoid duplicated issues, and check if the problem has already been solved.
  • 2. The issue must be labeled.
  • 3. Create an isolated and reproducible test case, where shows the steps that led to the problem occur.
  • 4. Share as much information as possible, like your operating system and version, the version of Billy, and other information that you think important.
